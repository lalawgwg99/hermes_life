# hermes_life brain repo

這個倉庫是「腦」的真相來源（Source of Truth）。
人類手動修改永遠優先；代理人只負責補充、整理與追加證據。

This repo is the source of truth for the brain.
Human edits always win. Agents append evidence to timelines.

## 快速開始 / Quick Start
1) 把新資訊丟進 inbox/
2) 由代理人整理到正確的類別（people/ companies/ ideas/...）
3) 夜間 Dream Cycle 自動補遺漏、修斷鏈、生成當日夢檔

## 核心設計 / Core Principles
- 上面是「Compiled Truth」：可修改、可修正的當前結論
- 下面是「Timeline」：只追加，不回寫，保留證據軌跡
- 所有新事實必須帶來源（source）
- 代理人必須先讀再寫
- 人類修改永遠覆寫機器結論

## 資料夾結構 / Folders
- people/      人物頁（一人一檔）
- companies/   公司頁（一公司一檔）
- ideas/       概念/論點（一概念一檔）
- meetings/    會議紀錄、逐字稿
- media/       文章/書/影片/Podcast 等素材
- originals/   原創想法與產出
- inbox/       未整理的輸入（Dream Cycle 會處理）
- meta/        規則、夢檔、系統設定
- templates/   統一格式模板

## 使用規則 / Rules
- Compiled truth 在上，timeline append-only 在下
- 每個新事實都要有來源（source）
- 代理人先讀再寫，避免覆蓋人類結論
- 任何自動寫入都需可回溯、可撤銷
